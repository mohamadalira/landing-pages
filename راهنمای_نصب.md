# راهنمای نصب و راه‌اندازی ربات

## مرحله 1: دریافت توکن ربات تلگرام

1. به ربات [@BotFather](https://t.me/BotFather) در تلگرام بروید
2. دستور `/newbot` را ارسال کنید
3. نام ربات خود را وارد کنید (مثلاً: My Landing Page Bot)
4. یوزرنیم ربات را وارد کنید (باید به `bot` ختم شود، مثلاً: `my_landing_bot`)
5. توکن دریافتی را کپی کنید (مثلاً: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## مرحله 2: دریافت توکن GitHub

1. به آدرس زیر بروید:
   ```
   https://github.com/settings/tokens
   ```

2. روی دکمه "Generate new token (classic)" کلیک کنید

3. نام توکن را وارد کنید (مثلاً: Landing Page Bot)

4. دسترسی‌های زیر را فعال کنید:
   - ✅ `repo` (Full control of private repositories)`
   - ✅ `workflow` (Update GitHub Action workflows)`

5. روی "Generate token" کلیک کنید

6. توکن را کپی کنید (⚠️ فقط یک بار نمایش داده می‌شود!)
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
## مرحله 3: تنظیم فایل config.py

فایل `config.py` را باز کنید و اطلاعات زیر را وارد کنید:

```python
# توکن ربات تلگرام
BOT_TOKEN = "توکن_دریافتی_از_BotFather"

# توکن GitHub
GITHUB_TOKEN = "توکن_دریافتی_از_GitHub"

# نام کاربری GitHub شما
GITHUB_USERNAME = "نام_کاربری_GitHub"

# نام ریپازیتوری (مثلاً: landing-pages)
GITHUB_REPO_NAME = "landing-pages"

# ایدی تلگرام برای پشتیبانی (بدون @)
SUPPORT_TELEGRAM_ID = "your_telegram_id"
```

## مرحله 4: نصب پکیج‌ها

در ترمینال یا CMD، دستور زیر را اجرا کنید:

```bash
pip install -r requirements.txt
```

## مرحله 5: اجرای ربات

```bash
python bot.py
```

اگر همه چیز درست باشد، پیام زیر را خواهید دید:
```
🤖 ربات در حال اجرا است...
```

## مرحله 6: تست ربات

1. به ربات خود در تلگرام بروید
2. دستور `/start` را ارسال کنید
3. اگر کانال اسپانسری تعریف نکرده باشید، ربات باید پاسخ دهد

## نکات مهم

- ⚠️ توکن‌ها را هرگز در گیت‌هاب آپلود نکنید!
- ✅ فایل `config.py` را به `.gitignore` اضافه کنید
- ✅ برای امنیت بیشتر، از متغیرهای محیطی استفاده کنید

## رفع مشکلات

### خطای "Invalid token"
- بررسی کنید که توکن ربات را درست کپی کرده باشید
- مطمئن شوید که توکن GitHub دسترسی‌های لازم را دارد

### خطای "Repository not found"
- بررسی کنید که نام کاربری GitHub درست باشد
- مطمئن شوید که توکن GitHub دسترسی به ریپازیتوری دارد

### ربات پاسخ نمی‌دهد
- بررسی کنید که ربات در حال اجرا باشد
- لاگ‌ها را بررسی کنید
